3 Simplify the polynomial expression:
6x + 12 / 3
Show your solution by factoring out the greatest common factor (GCF) and canceling out common terms.

MATHEMATICAL PROBLEM 1

1.) Remove the parentheses for the first two terms

3x³ − 5x² + 2x − 7 + 4x³ + 2x² − 3x + 5 − (2x³ − x² + 4x −2)

2.) Distribute the negative sign in the third term if there is a negative sign (-) in front of the parenthesis

3x³ − 5x² + 2x − 7 + 4x³ +2 x² − 3x + 5 − (2x³ − x² + 4x − 2)

2x³ → -2x³

−x² → x²

4x → -4x

−2 → 2

3x³ − 5x² + 2x −7 + 4x³ + 2x² − 3x + 5 − 2x³ + x²- 4x + 2

3.) Combine all of the like terms

3x³ − 5x² + 2x − 7 + 4x³ + 2x² − 3x + 5 − 2x³ +x² −4x + 2

3x³ + 4x³ + − 2x³ = 5x³

−5x² + 2x² + x² = 2x²

2x − 3x − 4x = -5x

−7 + 5 + 2 = 0

4.) Now form them all together again, which makes our final answer:

5x³−2x²−5x

MATHEMATICAL PROBLEM 2

We’re going to use the FOIL method

FIRST: Multiply the first terms together, which are 4x and 2x

4x ⋅ 2x = 8x²

OUTER: Multiply the outer terms, which are 4x and 3

4x ⋅ 3 = 12x

INNER: Multiply the inner terms, which are -5 and 2x

-5 ⋅ 2x = -10x

LAST: Multiply the last terms together, which are -5 and 3

-5 ⋅ 3 = -15

Form them all together and combine the like terms. After combining, what’s left will be our final answer.

8x² + 12x - 10x -15

8x² + 12x - 10x -15 (12x and -10x are like terms)

8x² + 2x -15

FINAL ANSWER: 8x² + 2x -15

MATHEMATICAL PROBLEM 3

1.) Factor 3 from the expression, since both 6 and 12 can be divided by 3, that makes 3 their GCF factor. If we divided 6x by 3, then it would be 2x. If we divide 12 by 3, then it would be 4. Hence, the top is 3 ( 2x + 4) because when you distribute them, it will turn back again to 6x + 12

3 (2x + 4) / 3

2.) Since there are 3 on the top and bottom, cancel both of them. This leaves us with 2x + 4, making that our final answer.

2x + 4

FINAL ANSWER: 2x + 4

MATHEMATICAL PROBLEM 1

1.) First, group the four terms into two groups. So that it would be a lot easier to solve

(x³ + 4x²) (− 9x − 36) = 0

2.) Factor out their common factors

x² (x + 4) - 9 (x + 4) = 0

3.) Based on the results, (x + 4) is their common factor because it’s present in both groups. So factor it out again

(x + 4) (x² - 9) = 0 → ( x² - 9 ) is considered as a difference of squares

(x + 4) (x + 3) (x - 3) = 0

4.) Solve for x, using the factored equation

x + 4 = 0

x = 0 - 4

x = -4

x + 3 = 0

x = 0 - 3

x = -3

x - 3 = 0

x = 0 + 3

x = 3

FINAL ANSWER: x = -4, -3, 3
